# Infomux

Infomux est une IA locale multi-personnalité sur PC/Mac.  
Elle fonctionne hors-ligne et propose 4 modes : Normal, Sans Filtre, Siri, Sombre.

## Installation

1. Installer Python 3.11+  
2. Installer les dépendances :
   pip install -r requirements.txt

## Lancer l'application

### Windows
Double-cliquer `build/build_windows.bat` → génère `Infomux.exe`

### Mac
Exécuter `build/build_mac.sh` → génère `Infomux.app`
